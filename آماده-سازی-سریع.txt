🔥 راهنمای فوری راه‌اندازی - فقط ۳ دقیقه!

مرحله ۱: ایجاد دیتابیس جدید در Access
-----------------------------------
1. Microsoft Access رو باز کن
2. New → Blank Database
3. اسم بذار: SchoolSystem.accdb
4. Create کن

مرحله ۲: اجرای کدهای اصلی
------------------------
1. برو به Visual Basic (Alt+F11)
2. Insert → Module  
3. کدهای فایل "کدهای-اصلی-VBA.txt" رو کپی کن
4. ذخیره کن با اسم Main_System

مرحله ۳: راه‌اندازی سیستم
-----------------------
1. پنجره Immediate رو باز کن (Ctrl+G)
2. تایپ کن: InitializeSystem
3. Enter بزن

✅ تمام! سیستم آماده است!
